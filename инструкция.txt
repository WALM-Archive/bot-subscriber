1. Установить программу python-3.12.4-amd64.exe, вставляя все галочки 
2. Распаковать 3 файла .py и файл .bat в отдельную папку
3. Открыть блокнотом config.py и прописать то, где есть описание(конкретно у BOT_API и admin), а также вставлять данные ВНУТРИ КАВЫЧЕК, ибо по итогу программа не запустится.
4. Запустите бота с помощью start.bat и напишите /start в чате с ботом
5. Пройдитесь по кнопкам, добавив ID основного телеграм канала и дополнительных телеграми канало, юзернеймы дополнительных телеграм каналов,
 приветствие и название основного телеграм канала. Бот объяснит, как формат ответа он требует, если необходимо. Если не объясняет как, то просто вставьте в чат данные. Также бот объяснит, для чего нужно. 
Дополнительно - для поиска ID тгк можно использовать бота GetMyID. ID начинается с -100 или со строчки Forwarded from chat ID: -1002178748026
		для поиска юзейрнеймов надо надо найти ссылку "t.me/юзернейм". Копируйте только юзернейм!.
		также вставляйте только правильные форматы ответов, иначе бот может сломаться и придеться распаковывать по новой данные архива(тобиш те самые 3 файла py с батником)
		если вы изменили ID основного телеграмм канала, то надо перезагрузить программу(о чем и напишет бот)
